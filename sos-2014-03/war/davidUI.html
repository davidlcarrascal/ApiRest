Instrucciones

Para consultar un año concreto: Pulsar consultar objeto y escribir en el campo año, el año de dicho objeto.
Para volver a la tabla desde cualquier campo pulsar consultar tabla, de esta forma tambien se actualiza. En ambos caso pulsar Enviar.

Para crear un nuevo objeto: Pulsar crear para poder rellenar todos los campos del objeto, a continuación pulsa Enviar.

Para modificar un objeto ya creado: Pulsar modificar para poder rellenar todos los campos del objeto a modificar (Los objetos se identifican por año), a continuación pulsa Enviar.

Para borrar un objeto: Pulsar borrary escribir en el campo año, el año de dicho objeto, a continuación pulsa Enviar.

Para borrar toda la lista: Simplemente pulsar en borrar todo y a continuación en Enviar.
